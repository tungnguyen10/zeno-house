# Source Text Quality

When writing source text, prioritize readability and language correctness.

## Required

- Never write Vietnamese without diacritics in source text.
- Source text includes: comments, user-facing labels/messages, seeded/demo text, and migration notes.
- Keep technical identifiers and code symbols in their original form.
- If high-quality Vietnamese phrasing is uncertain, use English instead of unaccented Vietnamese.

## Quick Self-Check

- Check Vietnamese diacritics for every user-facing string.
- Check spelling and natural phrasing in comments and copy.
- Keep terminology consistent within the same file/feature.
